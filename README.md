# Kenny Serpa — Software Engineer

Personal resume site built with Vite, served from the `gh-pages` branch.

## Local development

```bash
npm install
npm run dev
```

## Build and validate

```bash
npm run build
npm run validate
```

## Deployment

The site deploys automatically via `.github/workflows/deploy.yml` to the `gh-pages` branch. GitHub Pages is configured to serve from **Deploy from a branch** → `gh-pages` → `/ (root)`.
